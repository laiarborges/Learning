Lesson 1: Introduction to HTML

HTML (HyperText Markup Language) is the backbone of web development. It structures the content on the web.

<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <p>Hello, world!</p>
  </body>
</html>

Lesson 2: HTML Tags

HTML uses tags to define content. Some common tags include <p> for paragraphs and <h1> for headings.
